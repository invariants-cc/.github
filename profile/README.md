# invariants.cc

*Notes on building infrastructure-grade systems — with AI as both a tool and a component.*

This organization hosts the companion code for [invariants.cc](https://invariants.cc).

## What's here

Each repository is a sanitized public snapshot of a project discussed in a post. The posts are self-contained and carry their own argument; the code is here for readers who want to see the artifact behind it. The examples are deliberately small — the transferable part is the design, not the scale.

## How this work is made

I build and write with AI assistants in separate, cross-reviewing roles under my supervision: one implements, another reviews its work, and I make the decisions. Details on the [colophon](https://invariants.cc/colophon).
